Setting a file over to Github!

This is a tutorial on how to set a repository and attach it to an active center for Github!

  1. First: Open Git Bash, this is your typical command line running under UNIX.
  2. Then initalize Git Bash by typing: "git init"
  3. Then activate the system to the directory, by doing:"git checkout -b gh-pages.
  4. Then do "Git Status" to check how it is doing.

Commiting actions

  1. First, add git add "file-name"
  2. Then "git status" again to check on the files
  3. then "git commit -m "message"
  4. Then "git status" again

Setting up Github for these files

  1. Setup a new repository on Github's website
  2. Then copy the first line given to you to bring pre-existing data
  3. then type "git push -u origin gh-pages"

And there you go!
